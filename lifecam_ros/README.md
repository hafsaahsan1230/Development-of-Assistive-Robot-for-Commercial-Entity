LifeCam ROS

A ROS Noetic package developed for integrating a Microsoft LifeCam USB camera with robotic systems. The package publishes real-time video streams to ROS topics, enabling computer vision, monitoring, image processing, and autonomous robotics applications.

-Features

* USB camera integration with ROS Noetic
* Real-time image streaming
* ROS image topic publishing
* Compatible with OpenCV-based vision applications
* Suitable for robotics perception and monitoring systems

-Technologies Used

* ROS Noetic
* Ubuntu Linux
* C++
* Python
* OpenCV

-Applications

* Robot vision systems
* Object detection and tracking
* Remote monitoring
* Image acquisition for machine learning
* Assistive robotics projects

-Project Structure

* `launch/` – Launch files for camera initialization
* `src/` – Source code
* `scripts/` – Python nodes and utilities
* `config/` – Configuration files
* `package.xml` – ROS package configuration
* `CMakeLists.txt` – Build configuration

-Usage

Build the package inside a catkin workspace and launch the camera node using the provided launch files.

-Author

Developed as part of robotics and assistive robot research and development projects using ROS Noetic.
